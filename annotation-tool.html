üìã Installation Guide

Windows

  1. 1. Download the .msi installer
  2. 2. Run as administrator
  3. 3. Follow the installation wizard
  4. 4. Install .NET 6.0 if required
  5. 5. Launch from Start Menu

macOS

  1. 1. Download the .dmg file
  2. 2. Open the disk image
  3. 3. Drag app to Applications
  4. 4. Allow in Security preferences
  5. 5. Launch from Applications

Linux

  1. 1. Download the AppImage
  2. 2. Make it executable
  3. 3. Run: chmod +x OpenHOI.AppImage
  4. 4. Double-click to launch
  5. 5. Optionally integrate with desktop
